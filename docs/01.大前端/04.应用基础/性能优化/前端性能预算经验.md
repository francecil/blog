1. 性能指标（推荐）：
    1. 通用(pct75)：LCP < 2.5s; FMP < 2s; FCP < 1.5s
    2. 活动类项目：FCP < 1s, LCP < 1.5s
    3. 弱网情况下：FCP < 2s
2. 产物体积：
    1. 具体场景具体分析，大项目需要做好拆包和延迟加载
    2. H5 项目建议值：关键 JS + CSS 体积 < 200KB; 图片资源 < 2M
3. 接口响应：首屏关键接口耗时 < 500ms
4. 规则评分：Lighthouse 评分、pagespeed
    1. 性能得分 > 80 分
